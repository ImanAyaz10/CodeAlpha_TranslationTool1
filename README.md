# CodeAlpha_TranslationTool1
🌍 AI Language Translator (Groq + LLaMA 3.3)
Hugging face 🔗: [imanayaz10/translation-tool-CodeAlpha](https://huggingface.co/spaces/imanayaz10/translation-tool-CodeAlpha)

This project is an AI-powered Language Translation Tool developed using Python and Gradio. It integrates Groq’s LLaMA 3.3 large language model via API to perform intelligent and context-aware translation between multiple languages such as English, Urdu, Korean, French, and German.

The application allows users to:

Enter custom text

Select source and target languages

Receive instant translated output

The system demonstrates:

API integration

Large Language Model (LLM) usage

Prompt engineering

Real-time text processing

Interactive frontend development using Gradio

The project is deployable on Hugging Face Spaces, making it accessible as a web application.
